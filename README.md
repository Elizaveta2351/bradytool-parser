Парсер каталога bradytool.ru
Этот проект создан для сбора данных с сайта [bradytool.ru](https://bradytool.ru). Парсер собирает информацию о товарах и сохраняет её в CSV-файл для дальнейшего анализа.

📌 Задача (что хотел заказчик)

Собрать актуальные данные по товарам:
- Раздел каталога
- Код товара
- Производитель
- Цена
- Наличие на складе
- Прямая ссылка на товар
- Описание

Эти данные нужны для мониторинга ассортимента и цен конкурентов.

🔧 Что было сделано

1. Написан парсер на Python с использованием библиотек **BeautifulSoup** и **Requests**.
2. Скрипт обходит все страницы каталога по всем разделам.
3. Собранные данные структурированы и сохранены в CSV-файл.
4. Проведён первичный анализ данных с помощью **Loginom**.

🛠 Стек технологий

- Python 3.9
- BeautifulSoup4
- Requests
- CSV (встроенный модуль)
- Loginom (аналитика)
